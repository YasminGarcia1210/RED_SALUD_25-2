# RED_SALUD_25-2
⚕️ Proyecto de automatización del proceso de facturación en salud para la Red de Salud del Oriente, enfocado en la identificación y gestión de glosas mediante inteligencia artificial.
Incluye un prototipo funcional que automatiza la validación de facturas, prioriza correos y reduce reprocesos administrativos en E.S.E. públicas.
